# Fintech amplification questions

## objective
Qual é o output financeiro específico (DRE, fluxo de caixa, cap table, valuation)?
_Example:_ DRE projetada 18 meses + cap table pós-rodada.

## audience
Para quem (founder interno, investidor, banco, auditoria)?
_Example:_ Investor update pra rodada seed; sofisticação média.

## constraints
Sample size de dados, fontes (BACEN, B3, papers, dados próprios), regulação?
_Example:_ Dados próprios últimos 12m; tudo em BRL; compliance BACEN.

## examples
Tem modelo de referência que já usou e gostou (a16z templates, etc)?
_Example:_ Template da a16z "B2B SaaS" mas adaptado para fintech brasileira.

## scope
Cenários (base, conservador, otimista)? Sensitivity? Stress test?
_Example:_ 3 cenários + sensitivity em CAC e churn. Sem stress test.

## success_criteria
Vai usar para qual decisão (raise, hire, kill, double-down)?
_Example:_ Decidir se contrata 5 sales ou postpõe 6 meses.
